# 智能门窗与车库控制系统 (Intelligent Doors, Windows & Garage Control)

本项目包含用于物联网相关的智能门窗、车库控制系统硬件与软件参考资料，基于 STM32F103C8T6 核心板及 ESP8266-01S 无线模块。

## 项目结构
- **stm32**: 包含基于 STM32 的自动感应门及相关控制代码例程 (如步进电机、语音播报等)。
- **温湿度显示系统**: 包含温湿度、烟雾报警以及风扇、照明控制等源码与硬件设计资料，支持对接阿里云 IoT 智能 APP。
- **数据**: 包含所用硬件模块（STM32、ESP8266、各种传感器等）的开发文档、数据手册软件工具等备份材料。

## 硬件模块指南
1. **STM32F103C8T6**：系统主控核心，用于读取传感器和控制执行机构。
2. **ESP8266-01/01S**：Wi-Fi 模块，负责设备接入网络及与云端的通信。
3. **传感器和外设**：包括温湿度传感器 (DHT11 等)、烟雾传感器 (MQ 系列)、红外感应模块及步进电机等。
